RoleType

enum class RoleType

의미역(Semantic Role) 분석 표지를 담은 Enum class입니다. (ETRI 표준안)

Since
2.0.0

Enum Values

ARG0

동작주/행위자. 예: {"진흥왕은" 화랑도를 개편했다.}에서 <진흥왕은>.

ARG1

피동작주/대상. 예: {진흥왕은 "화랑도를" 개편했다.}에서 <화랄도를>, {"범인은" 사거리에서 발견되었다.}에서 <범인은>.

ARG2

시작점/수혜자. 예: {영희가 "철수에게서" 그 선물을 받았다.}에서 <철수에게서>.

ARG3

착점. 예: {연이가 "동창회에" 참석했다.}에서 <동창회에>.

ARGM_LOC

장소. 예: {친구들이 "서울에" 많이 산다.}에서 <서울에>.

ARGM_DIR

방향. 예: {달이 "서쪽으로" 기울었다.}에서 <서쪽으로>.

ARGM_CND

조건. 예: {"과세 대상 금액이 많을수록" 높은 세율을 적용한다.}에서 <많을수록>.

ARGM_MNR

방법. 예: {그는 "큰 소리로" 떠들었다.}에서 <소리로>.

ARGM_TMP

시간. 예: {진달래는 "이른 봄에" 핀다.}에서 <봄에>.

ARGM_EXT

범위. 예: {그 악기는 "4개의" 현을 가진다.}에서 <4개의>.

ARGM_PRD

보조서술. 예: {석회암 지대에서 "깔대기 모양으로" 파인 웅덩이가 생겼다.}에서 <모양으로>.

ARGM_PRP

목적. 예: {주나라의 백이와 숙제는 "절개를 지키고자" 수양산에 거처했다.}에서 <지키고자>.

ARGM_CAU

발생 이유. 예: {지난 밤 "강픙으로" 가로수가 넘어졌다.}에서 <강픙으로>.

ARGM_DIS

담화 연결. 예: {"하지만" 여기서 동, 서는 중국과 유럽을 뜻한다.}에서 <하지만>.

ARGM_ADV

부사적 어구. 예: {산의 능선이 "마치" 닭벼슬을 쓴 용의 형상을 닮았다.}에서 <마치>.

ARGM_NEG

부정. 예: {산은 불에 타지 "않았다."}에서 <않았다.>.

ARGM_INS

도구. 예: {"하얀 천으로" 상자를 덮었다.}에서 <천으로>.
