kr.bydelta.koala.ext / kotlin.CharSequence / assembleHangul

assembleHangul

fun CharSequence.assembleHangul(): CharSequence

주어진 문자열에서 초성, 중성, 종성이 연달아 나오는 경우 이를 조합하여 한글 문자를 재구성합니다.

사용법

Kotlin

// 왼쪽 문자열은 조합형 문자열임.
"까?ABC".assembleHangul() // "까?ABC"

Scala + koalanlp-scala

import kr.bydelta.koala.Implicits._
// 왼쪽 문자열은 조합형 문자열임.
"까?ABC".assembleHangul // "까?ABC"

Java

// 인자로 들어가는 문자열은 조합형 문자열임.
ExtUtil.assembleHangul("까?ABC") // "까?ABC"

Since
2.0.0

Return
조합형 문자들이 조합된 문자열. 조합이 불가능한 문자는 그대로 남습니다.
