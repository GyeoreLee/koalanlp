CanLabelSemanticRole

interface CanLabelSemanticRole : CanAnalyzeSentenceProperty<RoleTree>

의미역 분석(Semantic Role Labeling)을 수행하는 Interface입니다.

참고

의미역 결정은 문장의 구성 어절들의 역할/기능을 분석하는 방법입니다. 예) '나는 밥을 어제 집에서 먹었다'라는 문장에는 동사 '먹었다'를 중심으로

  • '나는'은 동작의 주체를,
  • '밥을'은 동작의 대상을,
  • '어제'는 동작의 시점을
  • '집에서'는 동작의 장소를 나타냅니다.

아래를 참고해보세요.

  • Word.getRole 어절이 직접 지배하는 하위 의미역 구조 RoleTree를 가져오는 API
  • Sentence.getSemRoleTree 전체 문장을 분석한 의미역 구조 RoleTree를 가져오는 API
  • RoleTree 의미역 구조를 저장하는 형태
  • RoleType 의미역 분류를 갖는 Enum 값
